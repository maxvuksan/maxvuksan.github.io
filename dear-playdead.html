Notable Projects

Tiny Modules is a project exploring modular audio synthesis. It was developed in C++ with JUCE.

Snapshots is a game about organizing messy spaces. It was developed in Unity3d and released onto Steam.

Wonk Wonk is a physics based multiplayer game about building and competing through obstacle courses. It is currently being developed in Unity3d.

A notable challenge of this project has been synronizing the physics simulation over the network; To achieve this, the games physics uses a custom solution built upon a deterministic fork of Volatile. Because of the deterministic nature, each client runs a simulation locally and is able to reconstruct each players state using just movement inputs.
